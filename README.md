## üî• Features

- **Clear All** functionality

- **Backspace** functionality

- Key binding:
    - ***Escape*** -> Clear All
    - ***Backspace*** -> Clear last char
    - ***Enter*** -> Equals
    - ***0-9 & math operators*** -> Add it to equation
